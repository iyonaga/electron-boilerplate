# electron-boilerplate
A minimal boilerplate for an Electron + React application.

## Installation

### Clone this repository
```
git clone https://github.com/iyonaga/electron-boilerplate.git <project-name>
cd <project-name>
```

### Install dependencies
```
yarn
```

## Usege

### Run the App
```
yarn start
```

### Start development
```
yarn dev
```

### Packaging
```
yarn package
```
